📊 Mutual Fund Analytics Project

🚀 Overview

This project focuses on data ingestion and analysis of mutual fund data.
The goal is to collect, process, and analyze financial data to generate meaningful insights.

---

📅 Day 1: Data Ingestion (ETL)

✅ Tasks Completed:

- Created structured project folders
- Loaded CSV datasets using Pandas
- Performed basic data inspection (.shape, .dtypes, .head())
- Fetched live NAV data using API
- Converted JSON data to CSV format
- Stored raw data for further processing

⚙️ How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run data ingestion:

python data_ingestion.py

3. Fetch live data:

python live_nav_fetch.py
